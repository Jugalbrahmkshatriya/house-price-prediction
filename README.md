# House Price Prediction

## Problem
Predict house prices based on features like location, size, and number of bedrooms.

## Solution
A simple machine learning model (Linear Regression / Random Forest) to predict house prices.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn

## Roadmap
- Step 1: Load dataset and explore features
- Step 2: Clean and preprocess data
- Step 3: Train model and evaluate
- Step 4: Save model for future use
